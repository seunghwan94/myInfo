프로젝트 목표
- ChatGPT의 발전에 빠르게 대응하기 위해 관련 기술을 활용하여 블로그 포스팅 자동화 프로그램을 개발

프로젝트 계획
- API 키 및 블로그 기본 설정을 텍스트 파일로 저장
- main.py 실행 시 저장된 정보를 사용하여 GPT 기반 블로그 게시글을 작성
- 게시글 작성 완료 후 이미지를 첨부하여 티스토리에 게시글을 게시
- 게시 성공 여부를 Slack으로 전달
- BIOS 설정을 통해 컴퓨터가 특정 시간에 자동으로 켜지도록 설정
- 작업 스케줄러를 사용하여 main.py를 실행하고, 실행 완료 시간에 컴퓨터가 자동으로 꺼지도록 설정
- API 연동
 - ChatGPT : GPTs를 통해 커스텀 하여 블로그 게시글 작성
 - Tistory : 게시글 작성되면 올라가는 블로그 사이트
 - Unsplash : 주제에 맞는 이미지 저장하여 게시글에 첨부
 - Google 스프레드시트 연동 : 허용된 아이디만 프로그램을 사용 가능
 - Slack : 게시글 작성 성공/실패 시 슬랙으로 전달
